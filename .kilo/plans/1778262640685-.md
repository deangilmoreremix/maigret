# AI Creative Personalizer - Design System Alignment Plan

## Objective
Rewrite the personalizer module to perfectly match the VideoRemix.vip design system, using the exact color palette, typography, component patterns, and glassmorphism effects from the existing codebase.

## Current Issues Analysis

### Color Mismatches (PersonalizerDialog.tsx)
| Current (Wrong) | Should Be (From index.css) |
|---|---|
| `bg-purple-600` | `bg-primary-600` (#6366f1) |
| `bg-cyan-600` | `bg-accent` (#ec4899) or `bg-primary` |
| `bg-emerald-600` | `bg-success` (#10b981) |
| `border-purple-500/30` | `border-accent-secondary/30` (#8b5cf6) |
| `bg-gray-800/50` | `bg-bg-elevated` (#1a1a1a) or `glass-effect` |
| `text-gray-300` | `text-gray-300` (ok) |
| `text-gray-400` | `text-gray-400` (ok) |

### Missing Design Elements
1. **No glassmorphism effects** - Missing `glass-effect`, `backdrop-blur-sm`, `border-white/10`
2. **No gradient effects** - Missing `text-gradient`, `gradient-bg` usage
3. **Raw HTML elements** - Using `<button>`, `<input>`, `<textarea>`, `<select>` instead of UI components
4. **Typography mismatch** - Not using `font-display` (Outfit) for headings, `font-body` (Plus Jakarta Sans) for body
5. **No framer-motion spring transitions** matching AppDetailModal.tsx
6. **Missing hover/focus states** that match existing components

### UI Components Available (src/components/ui/)
- `Button` - with variants (default, destructive, outline, secondary, ghost, link)
- `Input` - styled input component
- `Textarea` - styled textarea (currently uses `border-gray-600 bg-gray-900/50`)
- `Label` - (need to check)
- `Dialog` - Radix Dialog with overlay and content
- `Select` - (need to check)
- `Card` - (need to check)

```typescript
import { Button } from '@/components/ui/button';
```

#### 1.2 Fix Color Scheme
Replace all color classes to match index.css variables:

**Background colors:**
- `bg-gray-900` → keep (matches dark theme)
- `bg-gray-800/50` → `bg-[#1a1a1a] bg-opacity-50` or `glass-effect`
- Add `backdrop-blur-lg` for glassmorphism

**Primary actions (buttons):**
- `bg-purple-600` → `bg-primary-600` (#6366f1)
- `bg-purple-700 hover:bg-purple-700` → `bg-primary-600 hover:bg-primary-700`
- Add gradient `bg-gradient-to-r from-primary-600 to-primary-800` for primary buttons (matching AppDetailModal)

**Accent colors:**
- `bg-cyan-600` → `bg-accent` (#ec4899) or `bg-primary`
- `text-cyan-200` → `text-accent-foreground` or `text-white`

**Success colors:**
- `bg-emerald-600` → `bg-success` (#10b981)
- `bg-emerald-700` → darker shade of success

**Border colors:**
- `border-purple-500/30` → `border-accent-secondary/30` (#8b5cf6)
- `border-purple-500/20` → `border-white/10` (using glass-effect pattern)

#### 1.3 Apply Glassmorphism
Add to main container:
```typescript
className="... glass-effect backdrop-blur-lg border-white/10"
```

Add to sidebar and right panel:
```typescript
className="... bg-[#1a1a1a] bg-opacity-50 backdrop-blur-md"
```

#### 1.4 Use UI Components

**Buttons** - Replace raw `<button>` with `<Button>` component:
```typescript
// Before
<button className="px-4 py-2 bg-purple-600 hover:bg-purple-700...">

// After
<Button variant="default" className="bg-primary-600 hover:bg-primary-700">
  Personalize This
</Button>

// Ghost button variant for secondary actions
<Button variant="ghost" size="icon" onClick={onClose}>
  <X className="h-5 w-5" />
</Button>
```

**Inputs** - Replace raw `<input>` with `<Input>`:
```typescript
// Before
<input value={targetName} onChange={...} className="w-full bg-gray-800/50 border border-purple-500/30..." />

// After
<Input value={targetName} onChange={...} className="bg-gray-900/50 border-white/10 text-white" />
```

**Textarea** - Replace raw `<textarea>` with `<Textarea>`:
```typescript
// Before
<textarea className="w-full h-32 bg-gray-800/50..." />

// After  
<Textarea value={manualNotes} onChange={...} className="bg-gray-900/50 border-white/10 text-white" />
```

**Select** - Style native select or use Select component with proper classes:
```typescript
<select className="w-full bg-gray-900/50 border border-white/10 rounded-lg p-2 text-white">
```

#### 1.5 Typography
Add font classes:
```typescript
// Headings (h1, h2, h3)
className="... font-display font-bold tracking-tight"

// Body text
className="... font-body"

// Example for step titles
<h2 className="text-2xl font-bold text-white font-display tracking-tight">
  AI Creative Personalizer
</h2>
```

#### 1.6 Gradient Effects
Add gradient text for branding:
```typescript
<h2 className="text-2xl font-bold text-gradient font-display">
  AI Creative Personalizer
</h2>
```

Add gradient buttons matching AppDetailModal:
```typescript
<Button className="bg-gradient-to-r from-primary-600 to-primary-500 hover:from-primary-500 hover:to-primary-400">
  <Play className="h-5 w-5" />
  Open App

```

#### 1.7 Animations
Match AppDetailModal framer-motion patterns:
```typescript
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  exit={{ opacity: 0 }}
  className="fixed inset-0 bg-black/80 backdrop-blur-sm z-50"
>
  <motion.div
    initial={{ scale: 0.9, opacity: 0 }}
    animate={{ scale: 1, opacity: 1 }}
    exit={{ scale: 0.9, opacity: 0 }}
    transition={{ type: "spring", damping: 25, stiffness: 300 }}
    className="bg-gray-900 rounded-2xl max-w-6xl w-full max-h-[90vh] overflow-hidden shadow-2xl glass-effect"
  >
```

#### 1.8 Proper Spacing & Layout
Match AppDetailModal spacing:
- Header: `relative h-64 bg-gradient-to-br from-primary-600 to-primary-800` (for top section)
- Content padding: `p-6`
- Section spacing: `space-y-6`
- Button gaps: `gap-3`
